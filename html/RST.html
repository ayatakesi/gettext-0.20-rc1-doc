16.1.2 Resource String Table

RSTとは、バージョン3.0.0より前のFree Pascalコンパイラーの、リソース文字列テーブルファイルのフォーマットです。RSJは、バージョン3.0.0以降のFree Pascalコンパイラーにより作成された、新たなリソース文字列テーブルファイルのフォーマットです。
